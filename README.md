# CycloneLens

[NL]
CycloneLens is een interactief webgebaseerd platform waarmee gebruikers tropische cyclonen op een overzichtelijke en gebruiksvriendelijke manier kunnen volgen. Het platform biedt inzicht in actuele cycloondata en stelt gebruikers daarnaast in staat om eigen observaties toe te voegen.

CycloneLens wordt ontwikkeld met behulp van C# en het ASP.NET MVC-framework, waarbij gebruik wordt gemaakt van een N-laagarchitectuur om de verantwoordelijkheden binnen de applicatie te scheiden. Voor de opslag en het beheer van data wordt een relationele database gebruikt op basis van Microsoft SQL Server, beheerd via SQL Server Management Studio (SSMS).

Het project wordt ontwikkeld als een individueel project.


[EN] 
CycloneLens is an interactive web-based platform that allows users to track tropical cyclones in a clear and user-friendly manner. The platform provides insight into current cyclone data and also enables users to add their own observations.

CycloneLens is being developed using C# and the ASP.NET MVC framework, employing an N-layer architecture to separate responsibilities within the application. A relational database based on Microsoft SQL Server is used for data storage and management, managed via SQL Server Management Studio (SSMS).

The project is being developed as an individual project.
